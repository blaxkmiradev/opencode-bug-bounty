# IoT Security Skill

## Overview
Specialized in IoT device security assessment, firmware analysis, and embedded systems.

## Commands

### /firmware-analysis
Analyze IoT device firmware.

### /device-scan
Scan IoT devices for vulnerabilities.

### /embedded-scan
Scan embedded systems.

### /uart-scan
Scan UART interfaces.

### /jtag-scan
Scan JTAG interfaces.

## Usage
```
/firmware-analysis firmware.bin
/device-scan 192.168.1.1
/embedded-scan target
/uart-scan /dev/ttyUSB0
/jtag-scan <device>
```

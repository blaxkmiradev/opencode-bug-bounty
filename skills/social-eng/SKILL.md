# Social Engineering Skill

## Overview
Specialized in OSINT reconnaissance, information gathering, and social engineering preparation.

## Commands

### /osint-scan
Perform comprehensive OSINT scan on target.

### /whois-lookup
Perform WHOIS lookup on domain.

### /email-harvest
Harvest email addresses from target domain.

### /recon
Perform reconnaissance on target.

### /domain-info
Gather domain registration information.

## Tools Used
- subdomain_enum.py - Subdomain enumeration
- dns_enumeration.py - DNS records
- enum_users.py - User enumeration
- osint - OSINT tools

## Usage
```
/osint-scan target.com
/whois-lookup example.com
/email-harvest company.com
/recon target.com
/domain-info example.com
```

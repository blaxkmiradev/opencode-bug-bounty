# DevOps Security Skill

## Overview
Specialized in DevOps and CI/CD security, Kubernetes, Docker, and infrastructure auditing.

## Commands

### /k8s-scan
Scan Kubernetes clusters for security issues.

### /docker-scan
Scan Docker containers and images.

### /ci-cd-audit
Audit CI/CD pipelines for vulnerabilities.

### /infra-sec
Perform infrastructure security assessment.

### /config-audit
Audit configuration files for security.

## Usage
```
/k8s-scan <cluster>
/docker-scan <image>
/ci-cd-audit <pipeline>
/infra-sec <target>
/config-audit <file>
```
